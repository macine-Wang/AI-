# AI智能招聘助手 - 使用说明

## 🎯 系统概述

AI智能招聘助手是一个全流程、智能化的招聘管理系统，通过AI技术提升招聘效率和质量。

## ✨ 核心功能

### 1. 智能简历筛选与解析 📄

**功能特点：**
- 批量上传简历（支持PDF、DOC、DOCX格式）
- AI自动解析简历信息（教育背景、工作经历、技能等）
- 根据岗位JD智能匹配候选人
- 自动生成匹配度评分（70-100分）
- 快速识别优质候选人

**使用流程：**
1. 进入"简历筛选"标签页
2. 输入招聘职位名称和岗位要求（JD）
3. 上传候选人简历（可批量上传）
4. 点击"AI智能解析简历"按钮
5. 查看解析结果和匹配度评分
6. 将合适的候选人添加到人才库

**匹配度评分说明：**
- 90-100分：优秀匹配（绿色）
- 80-89分：良好匹配（蓝色）
- 70-79分：基本匹配（黄色）
- <70分：匹配度较低（灰色）

---

### 2. 候选人管理 👥

**功能特点：**
- 完整的候选人信息库
- 多维度搜索和筛选
- 状态流转管理
- 标签分类系统
- 面试记录追踪
- 沟通历史记录

**候选人状态：**
- 🆕 新申请
- 🔍 筛选中
- 📅 面试中
- 📧 已发Offer
- ✅ 已入职
- ❌ 已拒绝

**管理操作：**
- 查看候选人详细信息
- 更新候选人状态
- 生成面试问题
- 安排面试
- 删除候选人

---

### 3. 智能面试辅助 💡

**功能特点：**
- 待面试候选人一览
- AI自动生成面试问题
- 面试安排和提醒
- 多轮面试记录
- 面试评估表
- 面试反馈整合

**使用流程：**
1. 进入"面试辅助"标签页
2. 选择待面试的候选人
3. 点击"生成问题"按钮，AI自动生成8个针对性面试问题
4. 点击"安排面试"设置面试时间、面试官、面试类型
5. 面试后记录评分和反馈

**AI生成的面试问题类型：**
- 项目经验相关
- 技术能力考察
- 团队协作能力
- 职业规划
- 问题解决能力
- 学习能力
- 应变能力
- 动机和期望

---

### 4. 沟通自动化 ✉️

**功能特点：**
- 预置邮件模板库
- 支持变量替换
- 批量发送邮件
- 沟通记录追踪
- 模板自定义编辑

**邮件模板类型：**
1. **面试邀请** - 邀请候选人参加面试
2. **Offer发放** - 通知候选人获得Offer
3. **婉拒通知** - 礼貌地拒绝不合适的候选人
4. **面试提醒** - 面试前一天发送提醒

**支持的变量：**
- `{候选人姓名}`
- `{职位}`
- `{公司名称}`
- `{面试时间}`
- `{面试地点}`
- `{面试官}`
- `{薪资}`
- `{入职时间}`
- `{截止日期}`

**使用流程：**
1. 选择邮件模板
2. 编辑邮件主题和内容
3. 选择收件人（可多选）
4. 点击"发送邮件"

### 5. 数据分析与报表 📊

**关键指标：**
- 候选人总数
- 平均匹配度
- 平均处理时间
- 转化率

**招聘漏斗分析：**
- 新申请 → 筛选中 → 面试中 → Offer → 已入职
- 各阶段转化率可视化
- 识别招聘瓶颈

**招聘渠道效果分析：**
- 各渠道候选人数量
- 各渠道质量对比
- 渠道占比分析
- 优化渠道投放策略

- 智能问答系统

- 候选人对比

## 🚀 快速开始

1. **上传简历**
   - 进入"简历筛选"页面
   - 填写职位信息和要求
   - 上传候选人简历
   - 等待AI解析

2. **筛选候选人**
   - 查看匹配度评分
   - 选择优质候选人
   - 添加到人才库

3. **安排面试**
   - 进入"面试辅助"页面
   - 生成面试问题
   - 设置面试时间

4. **发送通知**
   - 进入"沟通管理"页面
   - 选择邮件模板
   - 发送面试邀请

5. **查看分析**
   - 进入"数据分析"页面
   - 查看招聘漏斗
   - 优化招聘策略

## 💡 提示和技巧

### 提高匹配准确度
- 详细填写岗位JD，包括具体技能要求

- 根据候选人背景调整问题
- 准备有针对性的追问

### 提高沟通效率
- 建立标准化的邮件模板
- 使用变量快速个性化
- 设置自动提醒

### 数据驱动决策
- 定期review招聘数据
- 识别优质渠道
- 优化招聘流程
