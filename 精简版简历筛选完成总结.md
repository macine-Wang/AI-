## 📊 技术实现细节

### 1. OCR识别流程
```typescript
文件上传 
  ↓
PDF验证
  ↓
OCR识别（阿里云）
  ↓
文本提取
  ↓
信息解析（正则+规则）
  ↓
数据结构化
  ↓
表格展示
```

### 2. 学历提取逻辑
```typescript
1. 扫描全文
2. 提取博士信息 → 如有，设为最高学历
3. 提取硕士信息 → 如无博士，设为最高学历
4. 提取本科信息 → 如无硕士和博士，设为最高学历
5. 保存所有学历信息到"说明"字段
```

### 3. 985/211判断
```typescript
function is985(university: string) {
  // 1. 查找大学排名数据
  const info = getUniversityInfo(university);
  
  // 2. 返回985属性
  return info ? info.is985 : false;
}
```

### 4. 排序实现
```typescript
// 大学排名排序
if (sortField === '院校排名') {
  comparison = a.院校排名 - b.院校排名;
}

// 学历排序
if (sortField === '学历') {
  const degreeOrder = { 
    '博士': 3, '硕士': 2, 
    '本科': 1, '专科': 0 
  };
  comparison = degreeOrder[b.学历] - degreeOrder[a.学历];
}
```
